# Visual Cryptography Project

## Academic Resources

1. **Visual Cryptography Revisited** - An in-depth look at visual cryptography techniques.
2. **Secret Sharing Schemes and Combinatorial Designs** - A study on different secret sharing protocols.
3. **Research papers and articles** from journals on cryptography and its applications.

## Deployment Guides

### Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your system.
- Clone the repository to your local machine.
  ```bash
  git clone https://github.com/Krishita17/Visual-Cryptography-ongoing.git
  cd Visual-Cryptography-ongoing
  ```

### Installation
- Navigate to the project directory.
- Install the dependencies using:
  ```bash
  npm install
  ```

### Running the Project
- Start the server:
  ```bash
  npm start
  ```
- Open your browser and go to `http://localhost:3000` to view the project.
